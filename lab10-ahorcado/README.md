# Juego del Ahorcado

Juego interactivo del ahorcado desarrollado en Angular. Contiene 20 palabras relacionadas con tecnología e ingeniería de sistemas, organizadas por categorías como Programación, Hardware, Redes, Seguridad, Web, IA y más.

## Características

- 20 palabras tecnológicas con categorías
- Dibujo del ahorcado en ASCII
- Teclado virtual y soporte para teclado físico
- Pistas disponibles durante la partida
- Contador de victorias, derrotas y récord consecutivo
- Corazones visuales para las vidas
- Diseño responsive y animaciones
- Botón para rendirse

## Requisitos

- Node.js 18 o superior
- Angular CLI 21

## Instalación

```bash
npm install
```

## Ejecutar

```bash
ng serve
```

Abrir en `http://localhost:4200/`.

## Construir

```bash
ng build
```

Los archivos de producción se generan en la carpeta `dist/`.
